구현 기능
(로그인)
JSP, Ajax등을 이용하여 프론트엔드를 구현한 후에 MVC 패턴을 이용해서 백엔드를 구현하고 xml 파일을 생성하여 sql문 작성하고 MySQL을 사용하고 후반에 들어서 아마존이라는 서버에 저장하여 사용하였습니다.

회원정보 수정에서는 프로필사진,비밀번호, 선호 카테고리들 만 수정 할 수 있다. 프로필 사진은 용량제한을 설정하여 용량을 초과하면 수정이 안된다. 비밀번호는 변경 버튼을 누르면 다른 창이 하나 더 뜨게 되는데 변경한 비밀번호를 확인 할 때 일치하지 않으면 변경이 안됩니다.
구현 기능
(영상 정보 변경)
회원정보수정과 마찬가지로 MVC패턴을 사용하여 구현하였고 DB에서 정보를 가져와서 정보들을 확인할 수 있도록 하고 후에는 아마존 서버에 저장하여 불러오도록 구현하였다.

동영상 정보 수정창에서 수정이 가능한 부분은 제목과 내용부분이며 동영상의 정보와 조회정보를 나누어서 사람들이 나의 동영상을 얼마나 많이 보고 좋아해주는지 편하게 알 수 있도록 나눠보았다.
구현 기능
(게시판 글 작성)
Mybatis를 이용하여 DB에 저장 되어 있는 게시물들을 가져와 보여주고, 관리자의 경우 notice의 값을 다르게 설정하여 공지 글을 올릴 수 있게 코딩하였습니다.
페이징을 통하여 한 화면에 게시글은 10개로 하고 공지 글은 제한이 없이 올릴 수 있게 하였습니다.
후기 등록 화면은 네이버 스마트에디터를 사용하여 글과 사진을 올리고 편집이 가능하게 하였습니다.
기존의 JSP로만 만들어져 있던 단순한 게시판을 jquery UI와 Bootstrap를 활용하여 디자인을 하였습니다.
회원정보수정과 마찬가지로 MVC패턴을 사용하여 구현하였고 DB에서 정보를 가져와서 정보들을 확인할 수 있도록 하고 후에는 아마존 서버에 저장하여 불러오도록 구현하였다.

대중적인 게시판 화면을 구현하여 누구나 쉽게 사용할 수 있도록 구현하였습니다 글의 삭제나 등록은 당사자만 가능하고 관리자는 게시판 관리를 위해 다 지울 수 있게 권한을 설정 하였습니다.
구현 기능
(게시판 글 수정)
MVC 모델을 이용하였고, 기존의 JSP로만 만들어져 있던 단순한 게시판을 jquery UI와 Bootstrap를 사용하여 디자인을 하였고, Mybatis를 이용하여 DB에 저장 되어 있는 해당 게시물의 내용을 가져왔습니다.

작성한 게시글을 수정 및 삭제가 가능하며 추천을 할 수 있는데 횟수 제한을 걸어서 추천과 비 추천은 한 번씩만 누르기가 가능하며 게시글을 작성한 당사자가 아니면 삭제 및 수정이 불가능합니다.
구현 기능
(게시판 댓글 작성)
MVC 모델을 이용하였고, 기존의 JSP로만 만들어져 있던 단순한 게시판을 jquery UI와 Bootstrap를 사용하여 디자인을 하였고, Mybatis를 이용하여 DB에 저장 되어 있는 해당 게시물의 내용을 가져왔습니다.

작성한 게시글에 댓글 및 대 댓글을 작성 가능하며 추천을 할 수 있는데 횟수 제한을 걸어서 추천과 비 추천은 한 번씩만 누르기가 가능합니다.
구현 기능
(게시판 댓글 삭제 및 수정)
MVC 모델을 이용하였고, 기존의 JSP로만 만들어져 있던 단순한 게시판을 jquery UI와 Bootstrap를 사용하여 디자인을 하였고, Mybatis를 이용하여 DB에 저장 되어 있는 해당 게시물의 내용을 가져왔습니다.

게시글에 댓글을 작성 후 수정 및 삭제가 가능하며 댓글을 작성한 당사자가 아니면 삭제 및 수정이 불가능합니다.
구현 기능
(관리자 페이지)
MVC 모델을 이용하여 회원,동영상,게시판 관리자 페이지를 만들었습니다
마스터 관리자는 회원에게 관리자 권한을 줄 수 있습니다.(권한을 받은 회원은 다른 회원의 올린 동영상 및 글 수정,삭제가 가능합니다.)
동영상 및 게시글 제목을 클릭 하면 좀더 상세한 정보를 볼 수 있습니다.
기존의 JSP로만 만들어져 있던 단순한 게시판을 jquery UI와 CSS를 적용했습니다.
